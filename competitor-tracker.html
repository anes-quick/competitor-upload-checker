How to get a YouTube API Key

  1. Go to console.cloud.google.com and create a free project.
  2. Navigate to APIs & Services → Library, search for YouTube Data API v3, and enable it.
  3. Go to APIs & Services → Credentials, click Create Credentials → API Key.
  4. Copy the key and paste it above. The free tier gives you 10,000 units/day — more than enough for this tool.
  5. To add a channel: paste the full YouTube channel URL (e.g. https://www.youtube.com/@ChannelName) in the field below.
Enter your YouTube API key in Settings to get started.
